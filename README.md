## Getting started

Clone & link the repository to your `.vmodules` directory.

```
git clone https://github.com/tobealive/bartender.git
cd bartender
ln -s $(pwd)/ ~/.vmodules/bartender
```

## Usage

Currently, please refer to the examples for usage.

### Run examples

```
v run examples/<file>.v
```
